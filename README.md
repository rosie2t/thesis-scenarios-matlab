# thesis-scenarios-matlab

Overview:
This repository contains MATLAB simulation scenarios developed as part of my Bachelor Thesis, focusing on the applications of Artificial Intelligence (AI) and Machine Learning (ML) in modern wireless communication systems. The goal of this project is to explore how AI techniques can improve performance, efficiency, and adaptability in dynamic wireless network environments.

About the scenarios:
### 1. Channel Estimation using CNN
Convolutional Neural Networks (CNNs) are used in order to compare them to Least Squares (LS) traditional method for estimating channels and to achieve accurate channel estimation

### 2. Dynamic Resource Allocation using Reinforcement Learning
A Q-Learning-based approach to optimize resource allocation between system performance and fairness

### 3. Path Loss Prediction using Machine Learning
Support Vector Regression (SVR) technique aims to predicts signal attenuation under realistic conditions

### 4. Intrusion Detection using Random Forest
Detects DDoS attacks in network traffic through supervised machine learning classification

### 5. Beamforming Optimization using DNN
Predicts optimal transmission angles and optimizing beamoforming technique
